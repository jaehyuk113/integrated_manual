만약 설정이 되어 있지 않다면 모바일에서는 기기의 설정 언어로, PC에서는 브라우저의 언어로 노출됩니다.'

## 1. 지원 언어
라이브폼은 총 17개 언어를 지원합니다.

| Language                | Code |
|-------------------------|------|
| Korean (한국어)         | `ko` |
| English (영어)          | `en` |
| Chinese (중국어)        | `cn` |
| Spanish (스페인어)      | `es` |
| Japanese (일본어)       | `jp` |
| Portuguese (포르투갈어) | `pt` |
| Vietnamese (베트남어)   | `vn` |
| Russian (러시아어)      | `ru` |
| Arabic (아랍어)         | `ar` |
| Finnish (핀란드어)      | `fi` |
| French (프랑스어)       | `fr` |
| Hindi (힌디어)          | `hi` |
| Indonesian (인도네시아어) | `id` |
| Persian (페르시아어)    | `pe` |
| Polish (폴란드어)       | `pl` |
| Thai (태국어)           | `th` |
| Turkish (트럭어)      | `tr` |

## 2. 언어 설정 방법
URL에 lang 파라미터 추가

`https://form.argosidentity.com/?pid={project_Id}&lang=en`

<Note>
파라미터 미설정 시 기본 동작:
- 모바일: 기기의 설정 언어로 표시
- PC: 브라우저의 언어 설정으로 표시
</Note>
