---
title: 'GET/Submission/userId'
description: '이 API를 사용하여 특정 user_id와 연관된 모든 제출 건(submissions)을 조회할 수 있습니다. 동일한 userid로 제출된 모든 KYC 데이터가 반환됩니다.'
---

## 1. 엔드포인트

```text GET/Submission/userId
GET https://rest-api.argosidentity.com/v3/submission
```
## 2. 인증

x-api-key 헤더에 API 키를 포함시켜야 합니다:

```text x-api-key
x-api-key: {yourAPIKey}
```


## 3. 요청 파라미터

<ResponseField name="userid" type="string" required="true">
 조회할 사용자의 고유 ID
</ResponseField>


## 4. 요청 예시

```curl GET/Submission/userId
curl --location --request GET 'https://rest-api.argosidentity.com/v3/submission?userid={userid}' \
--header 'x-api-key: {yourAPIKey}'
```
