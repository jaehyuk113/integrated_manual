## 2. 언어 설정 방법

URL에 lang 파라미터 추가, lang 파라미터의 값은 소문자로 기재

`https://form.argosidentity.com/?pid={project_Id}&lang=en`

<Note>
  파라미터 미설정 시 기본 동작:

  - 모바일: 기기의 설정 언어로 표시
  - PC: 브라우저의 언어 설정으로 표시
</Note>
