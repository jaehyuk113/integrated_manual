---
title: 'GET/Submission/email'
description: 'This API allows you to retrieve all KYC submission data associated with a specific email address. It provides KYC identity information, KYC results, AML results, rejection reasons and codes, retry reasons and codes, and more.'
---


## 1. Endpoint

```text GET/Submission/email
GET https://rest-api.argosidentity.com/v3/submission/applicant
```
## 2. Authentication

Include the API key in the x-api-key header:

```text x-api-key
x-api-key: {yourAPIKey}
```


## 3. Request Parameters

<ResponseField name="email" type="string" required="true">
The email address of the user whose submissions you want to retrieve.
</ResponseField>


## 4. Request Example

```curl GET/Submission/email
curl --location --request GET 'https://rest-api.argosidentity.com/v3/submission?email={email}' \
--header 'x-api-key: {yourAPIKey}'
```
