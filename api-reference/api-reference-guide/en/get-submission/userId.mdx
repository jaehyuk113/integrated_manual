---
title: 'GET/Submission/userId'
description: 'This API allows you to retrieve all submissions associated with a specific `user_id`. It returns all KYC data submitted by the same user ID.'
---

## 1. Endpoint

```text GET/Submission/userId
GET https://rest-api.argosidentity.com/v3/submission
```
## 2. Authentication

Include the API key in the x-api-key header:

```text x-api-key
x-api-key: {yourAPIKey}
```


## 3. Request Parameters

<ResponseField name="userid" type="string" required="true">
 The unique ID of the user whose submissions you want to retrieve.
</ResponseField>


## 4. Request Example

```curl GET/Submission/userId
curl --location --request GET 'https://rest-api.argosidentity.com/v3/submission?userid={userid}' \
--header 'x-api-key: {yourAPIKey}'
```
