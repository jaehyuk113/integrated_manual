---
title: "GET/Submission/email"
description: "이 API를 사용하여 특정 이메일 주소와 연관된 모든 KYC 제출 데이터(submissions)를 조회할 수 있습니다. KYC 신원 정보 데이터, KYC 결과, AML 결과, 거절 사유 및 코드, 재시도 사유 및 코드 등의 정보를 확인할 수 있습니다."
---

## 1. 엔드포인트

```text GET/Submission/email
GET https://rest-api.argosidentity.com/v3/submission/applicant
```

## 2. 인증

`x-api-key` 헤더에 API 키를 포함시켜야 합니다:

```text x-api-key
x-api-key: {yourAPIKey}
```

## 3. 요청 파라미터

<ResponseField name="email" type="string" required="true">
  조회할 사용자의 이메일 주소
</ResponseField>

## 4. 요청 예시

```curl GET/Submission/email
curl --location --request GET 'https://rest-api.argosidentity.com/v3/submission?email={email}' \
--header 'x-api-key: {yourAPIKey}'
```
