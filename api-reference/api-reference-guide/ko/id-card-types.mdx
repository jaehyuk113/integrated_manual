---
title: "ID 카드 종류"
description: "KYC 인증을 위한 API 호출에서 사용되는 신분증 유형을 설명합니다."
---

| idType                             | 설명                                                                                                         |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `government_id`                    | 정부에서 발급한 공식 신분증으로, 개인 신원을 확인하는 데 사용됩니다.                                                                    |
| `passport`                         | 정부에서 발급한 공식 여행 문서로, 국제 여행 시 사용되며 소유자의 신원과 국적을 증명합니다.                                                       |
| `drivers_license`                  | 특정 개인이 오토바이, 자동차, 트럭 또는 버스 등의 차량을 운전할 수 있도록 허가하는 공식 문서입니다.                                                 |
| `residence_permit`                 | 외국인이 일정 기간 동안 해당 국가에 거주할 수 있도록 이민국에서 발급한 공식 문서입니다.                                                         |
| `vehicle_registration_certificate` | 차량 등록 증명 문서로, 차량 및 소유자에 대한 세부 정보를 포함하는 공식 문서입니다.                                                           |
| `visa`                             | 여권에 부착된 공식 승인으로, 소유자가 특정 국가에 입국, 출국 또는 체류할 수 있음을 나타냅니다.                                                    |
| `aadhaar`                          | 인도 정부가 발급한 고유한 12자리 식별 번호로, 생체 정보와 인구 통계 데이터를 기반으로 합니다.                                                    |
| `pancard`                          | 인도 정부가 개인 및 단체에 발급하는 영구 계좌 번호(PAN) 카드로, 주로 세금 목적으로 사용됩니다.                                                  |
| `identification_card`              | 미국 주 정부에서 발급하는 신분증으로 미국 내에서 소유자의 신원과 국적을 증명합니다. 본 신분증은 신분증 국가 선택 시, 미국의 경우에서만 나타나며 사용자에게는 State ID로 표시됩니다. |
