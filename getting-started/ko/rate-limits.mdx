## API 요청 할당량 제한


당사의 API는 안정적인 서비스 제공과 리소스의 공정한 분배를 위해 요청 제한을 적용하고 있습니다. 모든 API 엔드포인트에 대해 다음과 같은 제한이 적용됩니다:

초당 최대 요청 수: 5,000 요청
일일 최대 요청 수: 100,000 요청



## 할당량 증가 요청

애플리케이션이 더 높은 요청 한도를 필요로 할 경우:

[지원 팀(Support Team)](Support 링크)에 문의하세요.
요청 한도 조정을 위한 사유와 예상 트래픽 세부사항을 포함한 요청서를 제출하세요.

한도 증가가 필요하신가요? 아래 정보를 제공해 주시면 신속히 검토하겠습니다:
증가 요청 한도: 현재 한도와 희망하는 새 한도를 명시해 주세요.
사유: 한도 증가가 필요한 구체적인 이유를 설명해 주세요.
예상 트래픽: 예상되는 일일/시간당 요청 수와 피크 시간대를 알려주세요.
기간: 한도 증가가 필요한 기간을 명시해 주세요 (임시 또는 영구).
기술 세부사항: 구현한 오류 처리 및 재시도 로직에 대해 간략히 설명해 주세요.
요청 검토 후 1-2 영업일 내에 답변 드리겠습니다. 긴급한 경우 고객 지원팀에 직접 연락 주시기 바랍니다.
