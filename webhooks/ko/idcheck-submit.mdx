## Prerequisites

ARGOS provides ID Check, a dedicated submission page for each project, accessible via a unique URL. To obtain the URL:

<Steps>
  <Step title="Log in to the dashboard.">
      <Card title="Open the Dashboard" icon="link" href="https://idcheck.argosidentity.com">
  </Card>

  </Step>
  <Step title="Navigate to Settings -> Access Management to locate the URL.">
  </Step>
</Steps>

## Options for Using ID Check in Verification Processes

ID Check supports two methods for completing identity verification

<img
  src="/images/getting-started_banner.png"
  alt="Hero Light"
/>

<Steps>
  <Step title="Using a PC">
      <img
    src="/images/getting-started_web.png"
    alt="Hero Light"
  />
  1. Open the ID Check URL on your computer.
  2. A QR code will be displayed.
  3. Scan the QR code with a mobile device to access the ID Check page in a mobile web browser.
  </Step>
  <Step title="Using a Mobile Device">
    <img
  src="/images/getting-started_mobile.png"
  alt="Hero Light"/>
  Link the URL to the event in your app where user ID verification is required. This will allow users to open the ID Check page directly.
  </Step>
</Steps>
