---
title: 'KYC Status'
description: ' KYC Status page provides a concise summary of the current verification state for each submission, including statuses like Incomplete, Pending, Approved, and Rejected, along with key details such as rejection reasons and retry attempts.'
---
| Code | Description |
|------|------|
| `Approved` | The submitted Liveform have been approved. |
| `Rejected` | The submitted Liveform  have been rejected. Reasons for rejection can be found in kyc.comment.|
| `Pending` | The submitted Liveform are under manual review. |
| `Incomplete` | Submission is incomplete due to insufficient KYC data.  |
