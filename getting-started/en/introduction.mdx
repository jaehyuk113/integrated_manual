Get started with ARGOS Identity's powerful identity verification solution. Build a secure and efficient ID check process in just a few simple steps.

## 1. What is Liveform?

Liveform is the identity verification form provided by ARGOS Identity's ID check. It is designed to effectively counter various types of fraud and comply with KYC (Know Your Customer) regulations.

### 1-1. Key Features of Liveform

- **Flexible Identity Verification Layers**: Adjust security levels by adding various authentication steps based on your needs.
- **Dynamic Risk Assessment**: Enable additional verification steps based on risk levels for more precise identity validation.
- **Regulatory Compliance**: Designed to meet various industry standards and regulatory requirements.
- **Adaptive Security**: Ensure safe and accurate identity verification while adapting to evolving fraud threats.

## 2. How to Use ID check Liveform

ID check can be used in two ways.

### 2-1. Starting on a PC

![liveform-pc](/images/liveform-mobile.png)

<Steps>
  <Step title="Open the ID check URL">
    Open the ID check URL on your computer.
  </Step>
  <Step title="Scan the QR Code">
    Scan the QR code displayed on the screen using your mobile device.
  </Step>

    Complete the ID Check process in the mobile web browser after scanning.
  </Step>

![liveform-app](/images/liveform-app.png)

  </Step>
  <Step title="Complete the Verification on Mobile">
    Complete the ID check process in the webview after scanning.
  </Step>
</Steps>

## 5. Submitting the Liveform

ARGOS ID check Liveform allows KYC submissions through a webpage. Customer account information and the ID check submission URL required for this service can be found in the dashboard.

### 5-1. Setting up the Liveform Link

<Warning>
  URL Guidelines: Do not alter the submission form URL. Modifications may prevent successful submissions.
</Warning>

#### 5-1-1. Methods to Integrate the Liveform

1. Embed the Liveform directly in an HTML iFrame.
2. Open the Liveform as a separate webpage.
3. Additional Features:

- [Query Strings and Tokens](/getting-started/en/liveform-url/querystring-and-token-guide)
- [Data Encryption and Decryption](/getting-started/en/encrypt-and-decrypt-data/overview)
- [Data Injection](/getting-started/en/liveform-url/data-injection-with-custom-fields)

<Note>
  Notes:

  1. If the URL parameter includes the email address, the submitter will not need to enter the email separately.\
     Example: Using a link with `&email={email_address}` skips the email input step.
  2. Email Notification: To notify submitters about the KYC process and results via email, include their email address in the URL parameter.
</Note>

### 5-2. Structure of the Liveform URL

- **Base Form URL**: `https://form.argosidentity.com/?pid={project_Id}`
- **URL with Email**: `https://form.argosidentity.com/?pid={project_Id}&email={email}`

Example: `https://form.argosidentity.com/?pid=argos5202&email=demo@argosidentity.com`

<Info>
  - **pid**: The `pid` in the Liveform URL is the unique ID of the client.
  - If you need to verify the `pid (project Id)`, please contact ARGOS Helpdesk.
</Info>
