## API 요청 할당량 제한


당사의 API는 안정적인 서비스 제공과 리소스의 공정한 분배를 위해 요청 제한을 적용하고 있습니다. 모든 API 엔드포인트에 대해 다음과 같은 제한이 적용됩니다:



<Info> 현재 ID Check API에 적용되는 API 사용 할당량은 다음과 같으며 변경될 수 있습니다.</Info>



| 제한 유형 | 할당량 | 설명 |
| :---: | :----: | :----: |
| 초당 최대 요청 수 (QPS) | 5,000 요청 | 1초 동안 처리 가능한 최대 API 요청 수 |
| 일일 최대 요청 수 (QPD) | 100,000 요청 | 24시간 동안 허용되는 총 API 요청 수 |





## 할당량 증가 요청


애플리케이션이 더 높은 요청 한도를 필요로 할 경우:
[CS 팀](https://support.argosidentity.com/hc/ko/requests/new)에 문의해주세요. 요청 검토 후 1-2 영업일 내에 답변 드리겠습니다. 
긴급한 경우 sales@argosidentity.com에로 문의해주세요.
