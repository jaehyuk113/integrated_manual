---
title: 'KYC 상태'
description: 'KYC 상태 페이지는 각 제출물의 현재 인증 상태를 간결하게 요약하여 제공합니다. 상태에는 불완전, 보류, 승인, 반려 등이 포함되며, 반려 사유와 재시도 횟수와 같은 주요 세부 정보도 함께 확인할 수 있습니다.'
---


| 코드         | 설명                                                   |
|--------------|--------------------------------------------------------|
| `Approved`   | 제출된 라이브폼이 승인되었습니다.                         |
| `Rejected`   | 제출된 라이브폼이 반려되었습니다. 반려 사유는 `kyc.comment`에서 확인할 수 있습니다. |
| `Pending`    | 제출된 라이브폼이 현재 수동 검토 중입니다.                  |
| `Incomplete` | KYC 데이터가 부족하여 제출이 완료되지 않았습니다.            |
