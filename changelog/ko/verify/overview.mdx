## Face Liveness API 출시 ##
Face Liveness API가 새롭게 출시되었습니다! 이 API는 얼굴 이미지가 실제 인물로부터 촬영된 것인지 확인해, 가짜 사진이나 영상을 이용한 사기 시도를 차단합니다. 이제 인증 과정에서 보다 안전하고 신뢰할 수 있는 검증을 할 수 있습니다.

### 주요 기능
<Steps>
  <Step title="얼굴 인증">
    제출된 얼굴 이미지가 실제 인물로부터 촬영된 것인지 즉각적으로 확인합니다. 이로써 가짜 이미지나 영상의 사용을 효과적으로 차단할 수 있습니다.
  </Step>
  <Step title="신뢰도 기반 점수 제공">
    검증된 얼굴의 신뢰도를 점수화하여 제공합니다. liveness score를 활용해 결과 임계값을 고객사의 특정 인증 요구사항이나 패턴에 맞게 유연하게 설정할 수 있습니다. 높은 점수일수록 검증 결과에 대한 신뢰성이 높아져, 정확한 의사결정을 지원합니다
  </Step>
  <Step title="사기 방지 및 보안 강화">
    가짜 얼굴을 이용한 인증 우회 시도를 차단하여 기업의 신원 검증 프로세스를 안전하게 보호합니다. 이를 통해 고객과의 신뢰도를 높이고, 데이터 보안을 강화할 수 있습니다.
  </Step>
</Steps>

## Face Compare API 업데이트

Face Compare API가 더욱 상세한 정보를 제공하도록 업데이트되었습니다!

새로운 응답 데이터:
`Landmarks`: 얼굴에서 감지된 주요 특징들의 위치.
`Pose`: 얼굴의 3D 방향 (좌/우, 상/하 회전 정보).
`Quality`: 이미지의 밝기와 선명도.

## API 변경 사항
### callbackUrl 매개변수 업데이트
요청 BODY 내 `callbackUrl` 매개변수가 선택사항으로 변경되었습니다. 이 변경으로 인해 API의 작동 방식이 다음과 같이 달라집니다:

1. **비동기 처리:** `callbackUrl`이 제공된 경우  
   결과가 지정된 `callbackUrl`로 전송됩니다. 자세한 내용은 **Callback 응답 예제**를 참조하세요.

2. **동기 처리:** `callbackUrl`이 생략된 경우  
   결과가 즉각적인 응답에 포함됩니다.

### 변경된 API 목록
다음 API가 이번 변경사항에 영향을 받습니다:
- [글로벌 ID 인식](https://dev.argosidentity.com/verify-id/global-id-recognition/global-id-recognition-api-1)  
- [ID 라이브니스](https://dev.argosidentity.com/verify-id/id-liveness/id-liveness-api-1)  
- [얼굴 비교](https://dev.argosidentity.com/verify-id/face-compare/face-compare-api-1)  

<Note>동기 응답에 대한 세부 정보는 **200 응답 문서**를 참조하세요. 비동기 응답의 경우 **Callback 응답 예제**를 확인하세요.</Note>  
</Update>
