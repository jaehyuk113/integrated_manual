## API 변경 사항
### callbackUrl 매개변수 업데이트
요청 BODY 내 `callbackUrl` 매개변수가 선택사항으로 변경되었습니다. 이 변경으로 인해 API의 작동 방식이 다음과 같이 달라집니다:

1. **비동기 처리:** `callbackUrl`이 제공된 경우  
   결과가 지정된 `callbackUrl`로 전송됩니다. 자세한 내용은 **Callback 응답 예제**를 참조하세요.

2. **동기 처리:** `callbackUrl`이 생략된 경우  
   결과가 즉각적인 응답에 포함됩니다.

### 변경된 API 목록
다음 API가 이번 변경사항에 영향을 받습니다:
- [글로벌 ID 인식](https://dev.argosidentity.com/verify-id/global-id-recognition/global-id-recognition-api-1)  
- [ID 라이브니스](https://dev.argosidentity.com/verify-id/id-liveness/id-liveness-api-1)  
- [얼굴 비교](https://dev.argosidentity.com/verify-id/face-compare/face-compare-api-1)  

<Note>동기 응답에 대한 세부 정보는 **200 응답 문서**를 참조하세요. 비동기 응답의 경우 **Callback 응답 예제**를 확인하세요.</Note>  
</Update>

## 새로운 기능
[텍스티파이 API](https://dev.argosidentity.com/verify/textify/introduction)가 새롭게 출시되었습니다! 데이터를 자동으로 추출하고 문서 워크플로를 획기적으로 개선하세요. 현재 미국 지역에서 지원되며, 텍스티파이는 팀이 중요한 문서 데이터를 쉽게 액세스할 수 있도록 돕습니다.
