## 1. 준비

- Node.js: 버전 16 이상
- CURSOR IDE: 최신 버전 (권장 버전 0.47.8 이상)

이미 설치되어 있는지 확인하세요:

```bash
node --version

npm --version
```

## 2. Terminal에서 MCP 설치

최초 1회만 아래 명령어를 사용해 MCP 서버를 설치합니다.

```bash
npx mint-mcp add argosidentity
```

어떤 플랫폼에 설치할 것인지 선택하라는 메시지가 표시됩니다.
