## 1. Preparation

Follow MCP quickstart guide, prepare to install Claude Desktop app and node
https://modelcontextprotocol.io/quickstart/user

## 2. Update Configuration in claude_desktop_config.json

Add the ARGOS manual configuration as follows

<CodeGroup>

```json claude_desktop_config.json (mac)
{
    "mcpServers": {
        "filesystem": {
            "command": "npx",
            "args": [
                "-y",
                "@modelcontextprotocol/server-filesystem",
                "/Users/{username}/Desktop",
                "/Users/{username}/Downloads"
            ]
        },
        "argosidentity": {
            "command": "node",
            "args": [
                "/Users/{username}/.mcp/argosidentity/src/index.js"
            ]
        }
    }
}{
```

``` claude_desktop_config.json (win)

    "mcpServers": {
```

Other browsers may also be supported, but the listed options offer the most stable performance.

## 3. Set up the MCP server

Install below command and set up the MCP server. It is only required first time.

```bash
npx mint-mcp add argosidentity
```

And launch claude desktop then you can search ARGOS manual in claude desktop app.
